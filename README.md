# Snowfall Desktop - Website

This repository contains the source code for the official landing page of the [Snowfall Desktop](https://github.com/BarredEwe/Snowfall) macOS application.

## Stack

-   **HTML:** Static content
-   **Tailwind CSS:** For styling (via CDN)
-   **Vanilla JS:** For language switching and the snow animation

## Deployment

The website is deployed by pulling the `main` branch on the server. No build step is required.
